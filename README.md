# Titanic-Survival-Prediction

**Goal:** Build a binary classification model to predict whether a passenger survived the sinking of the Titanic, using the classic Titanic dataset.

## Data Dictionary

| Variable     | Definition                                          |
|:-------------|:----------------------------------------------------|
| `survived`   | Survived? 0 = No, 1 = Yes *(target)*               |
| `pclass`     | Ticket class (1st, 2nd, 3rd)                        |
| `sex`        | Sex                                                 |
| `age`        | Age in years                                        |
| `sibsp`      | Number of siblings / spouses aboard                 |
| `parch`      | Number of parents / children aboard                 |
| `fare`       | Passenger fare                                      |
| `class`      | Ticket class (string form of `pclass`)              |
| `who`        | Man, Woman, or Child                                |
| `adult_male` | Boolean – adult male passenger                      |
| `alone`      | Boolean – travelling alone                          |
